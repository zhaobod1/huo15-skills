# 辉火云企业套件 v18.8.0 集成测试报告

| 项 | 内容 |

## 二、测试范围

1. **功能模块**：项目管理 / CRM / 联系人 / 报销 / 工时
2. **接口**：内部 API 124 个，外部 API（微信 / 钉钉 / 短信）8 个
3. **数据迁移**：v18.7 → v18.8 的 schema 升级与数据兼容
4. **安全**：身份认证、权限、防 SQL 注入、XSS

不在范围：移动端 H5（v18.9 再测）、第三方报表插件。

---

## 三、测试方法

- **功能测试**：用例 286 条，全自动化（pytest + Playwright）
- **接口测试**：Postman + Newman，全量回归
- **性能测试**：JMeter，1000 并发，30 分钟稳态
- **安全测试**：OWASP ZAP 主动扫描 + 人工渗透 4 小时
