# 辉火云企业版数据安全风险评估报告

## 一、评估范围

本报告对辉火云企业套件 v18.8.0 部署在生产环境的数据安全风险进行评估，覆盖：

- 数据存储：PostgreSQL 主库、Redis 缓存、对象存储
- 数据传输：API 网关、内部 RPC、第三方集成（微信 / 钉钉 / 短信）
- 访问控制：员工权限、第三方应用授权、API Key 管理
- 备份与恢复：每日全备 + 30 分钟增量

---

## 二、评估方法

1. 按 ISO 27001 / GB/T 22080 框架做控制项 mapping
2. 资产清单 + 威胁建模（STRIDE）
3. 风险矩阵（可能性 × 影响）= 风险等级
4. 现场访谈 + 技术抽样

---

## 三、风险识别

### R-001 数据库主备未自动切换

| 项 | 内容 |
|----|------|
| 资产 | 生产数据库 |
| 威胁 | 主库硬件故障 / 进程崩溃 |
| 漏洞 | 主备切换依赖人工介入 |
| 可能性 | 中（年均 1-2 次）|
| 影响 | 高（业务中断 1-3 小时）|
| 风险等级 | **高** |

### R-002 API Key 长期有效

| 项 | 内容 |
|----|------|
| 资产 | 第三方集成账号 |
| 威胁 | Key 泄漏 / 滥用 |
| 漏洞 | 部分 Key 三年未轮换 |
| 可能性 | 中 |
| 影响 | 中 |
| 风险等级 | **中** |

### R-003 备份未做异地容灾

| 项 | 内容 |
|----|------|
| 资产 | 数据库备份文件 |
| 威胁 | 数据中心整体故障 |
| 漏洞 | 备份与主库在同一可用区 |
| 可能性 | 低（年均 < 0.1 次）|
| 影响 | 极高 |
| 风险等级 | **中** |

### R-004 员工权限审计缺失

| 项 | 内容 |
|----|------|
| 资产 | 员工账号 |
| 威胁 | 离职员工权限残留 |
| 漏洞 | 离职流程未联动权限回收 |
| 可能性 | 中 |
| 影响 | 中 |
| 风险等级 | **中** |
