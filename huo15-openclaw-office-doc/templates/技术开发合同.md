## 第二条 开发内容与技术指标

### 1. 开发内容

| 模块 | 主要功能 | 技术指标 |
|------|---------|---------|
| 用户端小程序 | 注册登录 / 浏览 / 下单 / 支付 / 订单管理 | P95 响应 < 300ms |
| 用户端 H5 | 同小程序 | P95 响应 < 500ms |
| 商家端 PC | 商品 / 订单 / 库存 / 对账 | 支持 10 商家并发 |
| 平台运营端 | 用户 / 商家审核 / 数据看板 | RBAC 权限控制 |

### 2. 技术架构

- 后端：Java 17 + Spring Boot 3.x + PostgreSQL 15 + Redis 7
- 前端：uni-app（小程序+H5）+ Vue 3 + Vite（商家端 / 运营端）
- 部署：阿里云 8C16G + Nginx + Docker

### 3. 性能指标

- 单日下单量峰值：≥ 1000 单
- 同时在线用户：≥ 500 人
- 系统月度可用性：≥ 99.5%
