# 文档范本（v7.4 起 15 类）

每个 markdown 范本是一份"看了就能改"的真实样例，覆盖 v7.4 新增的 15 类规范。
直接拷走、把里面的占位字段（公司名 / 客户名 / 编号 / 日期 / 金额）替换成自己的，
再用对应 preset 渲染即可。

| 范本文件 | 对应规范 | 推荐用法 |
|---------|---------|---------|
| [个人简历.md](个人简历.md) | `个人简历` | 紧凑单页，无文档壳 |
| [报价单.md](报价单.md) | `报价单` | 表格驱动，明显合计 |
| [新闻稿.md](新闻稿.md) | `新闻稿` | dateline + 主体 + 媒体联系人 |

## 怎么用

```bash
# 拷一份范本
cp ~/.openclaw/skills/huo15-openclaw-office-doc/templates/报价单.md \
   /tmp/我的报价单.md

# 编辑成自己的内容
vim /tmp/我的报价单.md

# 渲染（auto 会按标题命中 报价单 preset）
python3 ~/.openclaw/skills/huo15-openclaw-office-doc/scripts/create-word-doc.py \
  --output 我的报价单.docx \
  --title "软件开发服务报价单" \
  --content @/tmp/我的报价单.md
```

也可以让 LLM 直接基于范本生成，先把范本贴给它做参考，再让它按你的具体需求改写。
