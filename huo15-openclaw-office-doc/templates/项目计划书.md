# 辉火云 v19.0 大版本升级项目计划书

## 一、项目背景

辉火云企业套件已迭代至 v18.8.0，目前支撑 50+ 中小企业客户。市场需求与技术演进推动我们启动 **v19.0 大版本** 升级，目标：

- 接入 Anthropic Claude 4.7 / 4.6 多模型路由
- ERP / CRM 全模块前端切到 Vue 3 + Vite
- 引入 Plugin Marketplace 让客户自定义流程
- 数据库平滑迁移到 PostgreSQL 16 + Patroni 主备

---

## 二、项目目标

| 维度 | 目标 | 衡量指标 |
|------|------|---------|
| **业务** | 客户数从 50 → 200 | 季度新签 ≥ 30 |
| **性能** | API P99 < 100ms | 监控数据 |
| **可用性** | SLA 99.95% | 月度可用性 |
| **生态** | 上线 10 个第三方插件 | Marketplace 数据 |

---

## 三、项目范围

**包含**：
1. 多模型路由层（claude-opus-4-7 / claude-sonnet-4-6 / claude-haiku-4-5）
2. 前端框架升级（Vue 2 → Vue 3 + Vite）
3. Plugin Marketplace（前后端 + 计费）
4. 数据库迁移（PostgreSQL 14 → 16 + Patroni）
5. 文档与培训

**不包含**：
- 移动端原生 APP（v19.5 再启动）
- 跨境部署（v20.0 再讨论）
