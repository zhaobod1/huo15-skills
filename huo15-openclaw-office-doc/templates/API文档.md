# 辉火云开放平台 API 文档

## 一、概述

辉火云开放平台为合作伙伴提供 RESTful API，覆盖项目管理、CRM、联系人、报销四个核心模块。

**Base URL**：`https://api.huo15.com/v1`
**协议**：HTTPS
**编码**：UTF-8
**响应格式**：JSON

---

## 二、鉴权

所有 API 调用需在 HTTP Header 携带：

```
Authorization: Bearer <ACCESS_TOKEN>
X-Huo15-Tenant: <租户ID>
```

`ACCESS_TOKEN` 通过 `/auth/token` 获取，有效期 2 小时；过期后调用 `/auth/refresh` 刷新。

---

## 三、通用响应格式

```json
{
  "code": 0,
  "message": "ok",
  "data": { ... },
  "request_id": "req-xxxx-xxxx-xxxx"
}
```

`code = 0` 表示成功；非 0 表示失败，对应错误码见 §六。

---

## 四、项目管理接口

### POST /projects

**说明**：创建项目

**请求体**：

```json
{
  "name": "辉火云 v19.0 升级",
  "description": "大版本升级，含多模型路由 + Vue 3 迁移",
  "owner_id": "user-001",
  "start_date": "2026-05-01",
  "end_date": "2026-10-15",
  "tags": ["v19", "upgrade"]
}
```

**响应**：

```json
{
  "code": 0,
  "data": {
    "id": "proj-2026-001",
    "name": "辉火云 v19.0 升级",
    "status": "planning",
    "created_at": "2026-04-27T10:30:00+08:00"
  }
}
```

### GET /projects/{id}

**说明**：查询项目详情

**路径参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| `id` | string | 项目 ID |

**响应示例**：见 POST 返回。

### PATCH /projects/{id}

**说明**：更新项目（部分字段）

### DELETE /projects/{id}

**说明**：删除项目（软删除）

## 五、CRM 接口

### POST /customers

**说明**：创建客户

**请求体**：

```json
{
  "name": "与境出行",
  "contact": "张鹏",
  "phone": "138-0000-0000",
  "industry": "出行",
  "tags": ["重点客户", "出境游"]
}
```

### GET /customers

**说明**：客户列表（支持分页与筛选）

**Query 参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| `page` | int | 页码，默认 1 |
| `size` | int | 每页条数，默认 20，最大 100 |
| `industry` | string | 行业筛选 |
| `tag` | string | 标签筛选 |

---

## 六、错误码

| code | 含义 | HTTP 状态 |
|-----:|------|----------:|
| 0 | 成功 | 200 |
| 40001 | 参数缺失 | 400 |
| 40002 | 参数格式错误 | 400 |
| 40101 | 鉴权失败（token 无效）| 401 |
| 40102 | token 过期 | 401 |
| 40301 | 权限不足 | 403 |
| 40401 | 资源不存在 | 404 |
| 42901 | 调用频率超限 | 429 |
| 50001 | 服务内部错误 | 500 |
| 50002 | 第三方服务异常 | 502 |

---

## 七、限流

- 默认限流：每租户 1000 次 / 分钟
- 突发限流：5 秒内最多 50 次
- 命中限流返回 `429`，`Retry-After` Header 给出重试间隔
