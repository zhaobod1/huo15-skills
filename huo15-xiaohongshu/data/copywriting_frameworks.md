## 一、蕉下三大句式（颠覆 / 修辞 / 升维）

> 蕉下（Beneunder）轻量化户外品牌。
> 关键认知：**不卖防晒衣，卖"敢晒太阳的野心"**。

### 底层逻辑
**痛点 → 场景 → 方案 → 情绪共鸣**

四步任何一步缺位，文案就掉一个层级。

### 句式一：「不是……而是……」颠覆认知

**结构：** `不是 {旧认知 / 表面诉求}，而是 {新定义 / 深层价值}`

**为什么管用：** 否定句先推开旧认知，肯定句拉高产品到价值观层面。
读者读完，**身份认同**直接被升级 — 我以为我在买 X，原来我在拥抱 Y。

**蕉下范例：**
- 「不是害怕太阳，而是拥抱太阳」
- 「人不是去了户外，而是回到了户外」

**通用模板：**
```
不是 {功能/旧认知}，而是 {价值观/新身份}
不是 {害怕 / 逃避 / 应对} X，而是 {拥抱 / 找回 / 重新选择} X
不是 {别人觉得的} Y，而是 {自己想要的} Y
```

**自检：** 否定的那一面，读者真的有共鸣吗？肯定的那一面，是品牌真能给的吗？

---

### 句式二：修辞 / 比喻 — 参数变画面

**结构：** 把功能/参数寄生在自然意象上。

**为什么管用：** 数字 / 参数 / 技术词读了就忘；意象 / 比喻能直接进入读者脑中的画面，
留存远超 5 倍。

**蕉下范例：**
- 「风是最好的造型师」（衣服抗风性能 → 风的"造型服务"）
- 「像仙人掌一样，给肌肤加一层防晒层」（防晒 + 保湿 → 仙人掌一个意象完成）

**通用模板：**
```
{自然意象} 是最好的 {产品角色}
像 {生物 / 自然现象} 一样 {做产品的事}
{产品} 是 {一个时间 / 一种感觉} 的 {延伸 / 暂停 / 延展}
```

**自检：** 比喻有没有过度陌生化？读者第一遍读得懂吗？

---

### 句式三：「每一……都是……」功能升维

**结构：** `每一 {产品交互的最小颗粒} 都是 {品牌承诺的情绪/状态}`

**为什么管用：** 把"使用产品"的日常动作 — 喝一口水、戴一次帽子、走一步路 —
拉高成品牌的"每一刻仪式"。日常 = 仪式感。

**蕉下范例：**
- 「每一口呼吸都是松弛的味道」（轻量化面料 → 呼吸感 → 松弛）
- 「让每一次去户外的决定，都变得笃定」（户外服 → 行动力 → 笃定感）

**通用模板：**
```
每一 {最小动作} 都是 {品牌情绪}
让每一次 {使用场景} 都 {品牌承诺}
每一 {数量词} {产品颗粒} 都装着 {价值观}
```

**自检：** "每一"后面接的是不是用户的真实日常动作？

---

### 蕉下 vs 人生尽兴指南：两种品牌姿态

| 维度 | 蕉下 | 人生尽兴指南 |
|------|------|------------|
| 角色 | 痛点爆破者（先否定旧的） | 情绪出口提供者（先创造新的空间） |
| 句式 | 对比 / 修辞 / 每一是 | 场景 / 留白 / 栏目化 |
| 卖什么 | 敢晒太阳的野心 | 暂停一下活成自己的许可证 |
| 姿态 | 「怕 X，就穿蕉下」直接给方案 | 「你的尽兴式是什么模样」让读者走到答案前 |
| 适合 | 功能型产品（户外/快消/工具） | 情感型 / 生活方式品牌 |

写文案前先选**姿态** — 你这次是要"爆破痛点"还是"提供出口"？

## 二、东东枪《文案的基本修养》核心概念

> 中信出版社 2019，东东枪著。99 篇小文讲透广告创意与文案之道。
> 这是商业文案的基本盘 — 比"句式套路"高一层。

### 1. 广告的定义
> **创作并传播内容，改变他人的看法或感受，以促成行为改变。**

注意：广告**不能直接改变行为**，只能改变"看法或感受"。
读者觉得 X 好（看法）/ 觉得 X 让我开心（感受）→ 才会去买。

### 2. AB 点理论（写每篇笔记前必问）

```
A = 消费者现在的看法 / 感受
B = 你希望他们改变后的看法 / 感受
广告 = 把人从 A 拉到 B 的桥
```

**实战自检：**
- 写之前：A 是什么？B 是什么？
- 写之后：读完这篇，A 真的变成 B 了吗？还是只看见品牌但情绪没变？

**常见错误：**
- ❌ 没想清楚 A，结果文案在打"假想敌"
- ❌ B 太大太空（"成为更好的自己"）— 没有具体感的 B = 没有 B
- ✅ A："周末就是用来恢复元气的" → B："周末是培育下一周好奇心的时间"

### 3. 核心体验（Key Experience）

**核心体验 = 品牌真正在售卖、被消费者真正买走的东西。**

经典例子：
- 电钻 ≠ 卖洞，卖**身份感 / 安全感 / 炫耀感**
- 蕉下 ≠ 卖防晒衣，卖**敢晒太阳的野心**
- 星巴克 ≠ 卖咖啡，卖**第三空间**
- 一个个人小红书号 ≠ 卖知识，卖**"和这个 IP 一起活着的感觉"**

**自检：** 读者真的来你这里只为信息？还是为了"和你一起的某种感觉"？

### 4. 洞察（Insight）

**洞察 = 未被发现或已被遗忘的真相。**

**只能"发现"，不能"发明"** — 你只能找到一个本来就存在的真相，不能编一个。

洞察存在于两个缝隙里：
- **认知与真相之间** — 大众以为 X，其实 Y
- **表达与认知之间** — 大众心里都觉得 X，但说不出来

读者看到一句好洞察的反应：「真相了」/「我也是这么觉得就是说不出」。

**写文案前问：** 这次你是在表达"已知"还是发现"未知"？

### 5. 品牌 = 固化的偏好

品牌的三种结果（按 ROI 升序）：
- **默认信赖** — 看到就觉得"这个不会差"
- **优先选择** — 同价位多家，先选这个
- **相对溢价** — 同款贵 30% 也愿意买

**对小红书号：**
- 默认信赖 = 标题不用太炸，读者也愿意点开
- 优先选择 = 同主题别人发了，他还愿意看你的
- 相对溢价 = 你出的合作 / 课 / 产品，粉丝愿意付钱

### 6. 品牌传播 vs 产品传播

| 传播类型 | 干什么 | 比喻 | 实战 |
|---|---|---|---|
| 品牌传播 | 让人更喜欢你 | 「为你唱首歌」 | 没产品的内容、人设笔记、价值观表达 |
| 产品传播 | 让人买你的东西 | 「给你房本车证」 | 测评、教程、好物推荐 |

**关键：** 品牌号 ≠ 每篇都在卖产品。**两种穿插着发**才能积累信任。

### 7. 心象 > 形象

**别用人口统计学定义读者**（25~35 岁女性 / 一线城市 / 月收入 1 万以上）。

**用"认知状态"定义：**
- 「觉得人生需要喘息但说不出口的人」
- 「想护肤但被 SK-II 价位劝退的人」
- 「装修过房子才发现家具排序错了的人」

心象越具体 → 文案越扎实。

### 8. Idea 金字塔

```
策略 Idea  — 决定方向（蕉下要"卖野心"不是"卖防晒"）
   ↓
创意 Idea  — 怎么打动人（"不是害怕太阳，而是拥抱"）
   ↓
执行 Idea  — 怎么呈现（具体文案 / 视觉 / 投放）
```

**最常见的错误：** 直接做执行 Idea，跳过策略和创意层。结果再漂亮的执行也是无根浮萍。

### 9. 创意碰撞法（Allen 在 2026-04-28 推荐的实战工具）

```
核心体验  ×  洞察  =  创意 Idea
```

举例：
- 核心体验：星巴克的"第三空间"
- 洞察：现代人在家工作焦虑，需要一个"被允许 What 不在家"的理由
- 创意 Idea：「让你的'不在家'有个去处」

**对小红书号：**
- 想清楚你的"核心体验"
- 找一个"未被表达但人人都有"的洞察
- 两者叠 → 让你独有的笔记 angle

## 三、对小红书的实战启示

### 写每篇笔记前的 5 问

1. **AB 点：** 读者现在怎么想？看完后怎么想？
2. **核心体验：** 你卖的不是内容，是哪种"感觉"？
3. **洞察：** 这一次你说的是已知还是未知？
4. **姿态：** 蕉下式爆破痛点 vs 尽兴式提供出口，这次走哪条？
5. **句式：** 蕉下三式 / Allen 五技法，本篇用哪个？

### 写完笔记后的 3 自检

1. 读者读完，A 真的变成 B 了吗？
2. 我这次是在卖产品，还是在唱歌？（两者比例失衡都不对）
3. 删掉 50% 字数，核心体验和洞察还在吗？（在 = 你写的是骨头）
