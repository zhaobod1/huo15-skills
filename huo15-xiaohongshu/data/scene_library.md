# 场景画面库 — 非节气日常画面

> 来源：Allen 教训 — "AI 写文案最大短板是场景库太窄，反复用同一批词
> （西瓜 / 蝉 / 冰棍 / 窗帘 / 傍晚的风）。"
>
> 这份库**不是节气**（节气见 seasonal_themes.md），是**人人都有过的日常瞬间**。
> 写文案前来这里挑 1~3 个画面入文，避免反复用同一批词。
>
> 共鸣度判断准则：你的奶奶 / 妈妈 / 邻居孩子 / 公司同事，他们都有过这个画面吗？

---

## 一、家中场景

### 厨房 / 餐桌
- 锅盖被蒸汽顶起来一下又落下
- 冰箱开门那 3 秒的灯
- 微波炉转盘卡住了又转动
- 倒水时杯沿那一圈水痕
- 收拾碗筷时筷子掉地上
- 切菜板上反复擦不掉的刀痕
- 茶包泡到发涨变形
- 早餐烤面包卡住又弹起来

### 客厅 / 沙发
- 沙发缝里掉的硬币 / 遥控器电池
- 抱枕被坐塌一个角
- 茶几玻璃上反复擦不掉的水印圆
- 地毯上猫毛 / 头发 / 灰尘的混合
- 电视屏幕上的指纹
- 窗台上那盆养不死也养不活的绿植

### 卧室 / 床
- 早晨闹钟响第一遍按掉的瞬间
- 被窝里手机滑落一下
- 枕头中间凹下去的形状
- 半夜踢飞被子又找回来
- 闹钟前 5 分钟自然睁眼
- 衣柜门关不严夹着衣角
- 化妆台抽屉里掉的耳环

### 卫生间 / 浴室
- 镜子上的雾 + 用手抹出一块清晰
- 牙膏快用完时挤的形状
- 洗澡时发现地漏堵了
- 毛巾杆上挂着的晾不干的小袜子
- 洗发水快没了瓶子歪着接
- 镜子里看到的自己刚睡醒

## 二、通勤 / 路上

### 地铁 / 公交
- 早八地铁里只有一只手能动的距离
- 包带被旁边人踩到
- 耳机线被门夹了一下
- 上班路上看见同一只猫
- 通勤时听同一首歌的副歌部分
- 闸机刷卡前 0.5 秒的迟疑
- 地铁玻璃倒映里自己的脸
- 雨天地铁里湿伞滴下的水痕

### 路上 / 电梯
- 红绿灯倒计时变 1 秒时迈出脚
- 电梯里和邻居装作没看见
- 雨天踩到积水溅自己一身
- 路过咖啡店那一阵咖啡香
- 听见后面有人小声讲电话
- 红绿灯下别人的伞蹭到自己
- 走路被一只狗盯着看
- 自动门没识别走快 0.5 秒

## 三、办公 / 工作

### 工位 / 会议室
- 会议室空调直吹脖子
- 屏保亮起那一瞬
- 工位上别人放的零食
- 鼠标垫起毛了一角
- 抽屉里被遗忘的口罩 / 纸巾
- 视频会议忘开摄像头那 3 秒
- 茶水间排队的咖啡机
- 工位上养了一年的多肉

### 加班 / 下班
- 加班到只剩自己工位灯亮
- 下班路上看到办公楼灭了大半
- 周五下午 5:30 同事的告别
- 周一早上电脑开机的卡顿
- 下班高峰电梯按到楼层亮一片
- 加班时楼下保安巡逻一圈
- 收工时关掉显示器那一刻

## 四、季节 / 天气（不限节气）

### 阳光 / 风
- 中午办公室落下一束斜阳到键盘上
- 风把书页翻过去一页
- 树荫下骑车，光在身上走
- 傍晚的风把头发吹到脸上
- 阳光晒到的地板有一只猫
- 风把帽子吹偏，没掉
- 太阳下山前最后那 10 分钟金光

### 雨 / 雾
- 雨打到伞上那种"咚咚咚"
- 阴天上班路上的湿凉空气
- 雾天看不清前 50 米
- 一下雨就觉得想吃面
- 雨后路面的水反光
- 屋檐滴水节奏不规则
- 玻璃窗上的雨痕从上往下

### 晨 / 夜
- 清晨第一口冷空气进鼻子
- 凌晨 4 点醒来翻个身
- 深夜厨房一杯水的声音
- 开灯前那 1 秒黑暗
- 凌晨手机屏幕的光
- 早上小区里有人晨练的咳嗽声
- 晚饭后散步的影子被路灯拉长
- 失眠时听到的远处车声

### 室内温度
- 关空调的那一瞬感觉热
- 冬天室内开暖气干到鼻子
- 夏天进空调房的"呼"一下
- 出门发现忘穿外套但回不去拿
- 浴室出来到客厅的温差

## 五、身体 / 感官

### 触觉
- 翻书页的纸边
- 衬衫领子和脖子摩擦
- 凉拖鞋打到脚后跟
- 戴久了眼镜在鼻梁上的印
- 把手插进毛衣口袋
- 摸到刚晒过的被子那种暖
- 抠手指甲缝
- 一直拿手机手腕开始酸

### 听觉

- 隔壁邻居家的电视声
- 清晨小区的鸟叫
- 远处工地的"咚咚咚"
- 楼下小卖部的关门声

### 嗅觉

- 老房子木头地板的味道
- 新书翻开的纸张味
- 同事身上香水的余味
- 电梯里前一个人留下的烟味
- 洗完澡用浴巾擦头发的味
- 地铁里有人吃了大蒜的味

### 视觉细节
- 衣服上一块洗不掉的油渍
- 钥匙串上多出来不知什么的钥匙
- 钱包里塞着的过期发票

## 六、关系 / 人际

### 家人
- 爸妈打电话第一句"吃饭了吗"
- 妈妈把不喜欢吃的菜夹给你
- 爸爸的脚步声从门外传来
- 视频通话挂掉后镜头里残留的家
- 兄弟姐妹偷穿你的衣服
- 老人家手机字号开到最大

### 朋友
- 朋友迟到 10 分钟时收的消息"快到了"
- 跟朋友吃饭最后一筷子的让来让去
- 微信群消息 99+ 时的犹豫

- 大学同学群突然冒出的婚礼通知

### 路人
- 路上被陌生人微笑了一下

## 七、独处 / 内心

### 独处时刻
- 一个人吃饭看手机的姿势
- 周日下午什么也不想做的那 1 小时
- 洗澡时唱歌的瞬间
- 通勤时戴上耳机进入自己的世界
- 厕所里多坐 5 分钟刷手机
- 散步时绕路多走 10 分钟
- 半夜想吃东西又懒得动

### 情绪瞬间
- 被冒犯了但当下没反应过来
- 微信打字打了又删
- 看到一条消息读了 3 遍
- 想哭但是哭不出来
- 突然想起一件 5 年前的小事

- 喝完一杯咖啡才意识到天黑了

## 怎么用本库

### 写文案前 30 秒：抽 3 个
1. 从相关类目里挑 3 个**人人都有过**的画面
2. 不要同一类目挑 3 个（避免单一感官堆砌）
3. 写进笔记的位置：开场 / 中段反差 / 结尾余韵

### 自检：抽到的 3 个画面够"具体"吗？
- ❌ "下午的风" → 太泛
- ✅ "中午办公室落下一束斜阳到键盘上" → 具体到能看见

### 长期：每周从这里"借"3 个不一样的画面
- 避免"同一个 AI 反复推荐同 5 个画面"
- 用过的画面在自己 baseline 里标一下
