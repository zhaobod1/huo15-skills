## 总纲：创作的第一性原理

**小红书文案的本质不是「写得好」，是「让对方看完后觉得自己被理解了」。**

一切技法从这个原点出发。如果读者看完没有「对，我就是这样」的瞬间，你的标题再炸、排版再漂亮，都是零。

**四句心法（写前 30 秒默念）：**

1. 「好文案不是写出来的，是留出来的。」— Allen 第一课
2. 「卖的是身份认同，不是商品本身。」— Allen 第二课
3. 「站文案里面读文案，不是站在外面分析。」— Allen 第三课
4. 「一流的糖衣，本身就是炮弹。」— 东东枪 041

## 第〇层：创作热身 — 动笔前 2 分钟

> 运动员上场前要拉伸，写文案前也需要把脑子从「分析模式」切到「感受模式」。
> 跳过热身直接写 = 用分析脑写感受文 = AI 腔 + 攻略型。

### 三步热身仪式

```
Step 1: 切换感官（30 秒）
  → 从场景库随机抽一个画面
  → 闭上眼真正感受它：温度、声音、光线、气味
  → 目的：从「想」切到「感」

Step 2: 找到情绪（30 秒）
  → 今天你想传递什么情绪？（不是主题，是情绪）
  → 回忆：上一次我感受到这种情绪是什么时候？
  → 目的：从编场景到捞记忆

Step 3: 对齐哲学（30 秒）
  → 默念四句心法
  → 自问：我是在邀请读者进入一个感受，还是在教她做一件事？
  → 目的：校准创作姿态
```

### 如果卡住了

- 去场景库随机翻一页，挑第一个打动你的画面
- 问自己：「关于这个画面，我最真实的感觉是什么？」
- 用那句真实的感觉作为第一句
- **不要想「读者想看什么」，先想「我真正想说什么」**

```bash
python3 scripts/warmup.py              # 完整三步热身
python3 scripts/warmup.py --quick      # 只看一个画面+情绪，直接动笔
python3 scripts/warmup.py --freewrite  # 场景 → 60 秒自由书写
```

## 第一层：创作原点 — 你为谁写？

### 1.1 用「心象」定义读者，不用「形象」

❌ 25-35岁女性 / 一线城市 / 月入1万+
✅ 「觉得人生需要喘息但说不出口的人」
✅ 「想对自己好一点但总把别人排前面的人」

**Why:** 人口标签只能描述「谁可能买」，心象描述「谁会产生共鸣」。共鸣驱动一切互动行为（评论×4 / 转发×4 / 关注×8）。

**自检：** 你定义的那个读者，你能在脑子里看见她今天下午在做什么吗？看不见 = 太模糊。

### 1.2 站文案里面感受，别站外面分析

写每句话的时候问自己：**「如果我是读者，读到这一行，我的手会停下来吗？」**

- 读者在刷小红书的时候是**放松模式**，不是学习模式
- 互动引导不是任务指令，是邀请语——「这里有个局，你可以来」
- 文案是情境，不是说服

### 1.3 场景共鸣的铁律

> 「场景共鸣不是找冷知识，是找人人都有过的共同记忆。」— Allen

| ❌ 冷知识（装文化，无共鸣） | ✅ 共同记忆（谁都有过） |
|---|---|
| 蝉要在地下七年才出土 | 树荫下骑车，光在身上走 |
| 古人夏天喝什么 | 风扇转头的嗡嗡声 |
| 西瓜原产自非洲 | 切开西瓜那一声脆响 |

## 第二层：创作支点 — 你要改变什么？

### 2.1 AB 点：定义你要改变的「看法或感受」

> 「广告只能改变看法和感受，不能直接改变行为。」— 东东枪 006

**写每篇笔记前必答：**

```
A = 读者现在的看法/感受：
B = 看完后我希望她有的看法/感受：
这座桥怎么搭？
```

**B 点的质量判断：**
- ❌ 「让她觉得我的产品好」→ 这是愿望，不是 B 点
- ❌ 「成为更好的自己」→ 太虚，没有具体感的 B = 没有 B
- ✅ 「让她觉得周末不是用来恢复元气的，是用来培育下周好奇心的时间」→ 具体、可感

### 2.2 核心体验：你不卖内容，你卖感觉

> 「核心体验 = 品牌真正在售卖、被消费者真正买走的东西。」— 东东枪 014

- 一个护肤号 ≠ 卖护肤知识，卖的是**「我值得被好好对待」的许可**
- 一个职场号 ≠ 卖职场技巧，卖的是**「我不是一个人在这种处境里」的陪伴**
- 蕉下 ≠ 卖防晒衣，卖的是**敢晒太阳的野心**

**自检：** 读者真的来你这里只为了信息？还是为了「和你一起的某种感觉」？

### 2.3 先做功课再动笔（Allen 红线）

**写任何产品/品牌前：**
1. 搜品牌官方宣传语、发布会、产品页
2. 理解品牌怎么定义自己（不是你怎么看它）
3. 提取 3-5 个官方核心意象
4. 判断品牌卖的是态度还是成分
5. 再做创意

❌ 跳过 = 从零发明产品 = 大概率 0 分。品牌自己在说什么 > 你觉得品牌应该说什么。

### 2.4 身份认同符号系统

> Allen 第二课：「卖的是身份认同，不是商品。」但身份认同**不是抽象的**，
> 它通过 5 类符号被读者识别——你写不出符号，读者就认不出「是不是同类」。

| 符号类型 | 示例 | 错误用法 | 正确用法 |
|---|---|---|---|
| **物** | 手冲壶 / 老式打字机 / 帆布袋 | 「我喜欢咖啡」 | 「桌上那只蓝月亮的手冲壶今天又凉了」 |
| **地** | 后厨 / 美院旁边的咖啡馆 / 凌晨的医院走廊 | 「我经常去咖啡馆」 | 「美院后门那家只放黑胶的店，下午三点没什么人」 |
| **行** | 把书脊朝外摆 / 在地铁里读纸质书 / 周日下午整理胶卷 | 「我爱读书」 | 「她书架最上面一层，书脊朝里——她说封面太花会扰乱睡眠」 |
| **时** | 早六点没人的湖边 / 周三的工作日午餐 / 立冬这天的下午茶 | 「我作息健康」 | 「立冬那天下午三点，我给自己泡了第一壶白茶」 |
| **语言节奏** | 短句 / 重复句 / 留白处的"嗯"/"吧" | 写得密不透风 | 「嗯。就这样。」「也可能不是。」 |

**自检（写完一段问自己）：**
- 这一段里至少有 1 个具体的「物 / 地 / 行 / 时」吗？
- 还是全是抽象形容词（「精致」「松弛」「治愈」）？

**符号 ≠ 道具堆砌：** 一段 80 字里塞 5 个符号 = 摆拍。**1-2 个真实细节 > 5 个炫耀符号。**
真实的符号是顺手提到的，不是用来证明身份的。

**自我推翻：** 你写的符号，是你真的拥有/做过的吗？如果不是，读者会闻出来——
读者比你想的敏感，**伪装的身份认同 = 反向身份认同**（让真同类反感）。

## 第三层：创作手艺 — 怎么写？

### 3.1 留白法则

> 「好文案不是写出来的，是留出来的。」— Allen

- 断句 + 留白，不是连续推演
- 金句独立成段，**不解释道理**
- 写完一段想：「读者读到这里能不能停一下，往里填自己的情绪？」

**技法：** 把「你应该」「你必须」「这是因为」删掉，换成「有时候」「也许是」「我自己是」。

### 3.2 五法：不说产品，说人的感受

| # | 手法 | ❌ 不要 | ✅ 要 |
|---|------|---------|-------|
| 1 | 用感受代配料 | 写配料表/成分 | 写吃下去/用上去的感觉 |
| 2 | 用场景代功能 | 说产品能干嘛 | 展示什么场景下用 |
| 3 | 用情绪代产品 | 以产品为主角 | 以读者情绪为主角 |
| 4 | 用过程代成分 | 列成分/参数 | 写从源头到成品的旅程 |
| 5 | 用比喻代描述 | 直接描述产品 | 用自然意象包裹产品 |

### 3.3 三大句式（蕉下体系）

**句式一：「不是……而是……」— 颠覆认知**
先否定旧预设，再拉高到价值观。
- 「不是害怕太阳，而是拥抱太阳」
- 公式：不是 {旧认知/表面诉求}，而是 {新身份/深层价值}

**句式二：修辞/比喻 — 参数变画面**
把产品功能寄生在自然意象上。
- 「风是最好的造型师」
- 公式：{自然意象} 是最好的 {产品角色}

**句式三：「每一……都是……」— 功能升维**
把日常小事拉高成品牌体验。
- 「每一口呼吸都是松弛的味道」
- 公式：每一 {最小动作} 都是 {品牌情绪}

### 3.4 洞察：只能发现，不能发明

> 「洞察 = 未被发现或已被遗忘的真相。」— 东东枪 020

洞察存在于两个缝隙里：
- **认知与真相之间** — 大众以为 X，其实 Y
- **表达与认知之间** — 大众心里都觉得 X，但说不出来

**好的洞察说出来，读者的反应是「真相了」「没法反驳」「我也是这么觉得就是说不出」。**

### 3.5 创意碰撞公式

```
核心体验 × 洞察 = 创意 Idea
```

- 核心体验：尽兴指南卖的是「停下来，此刻就是最好的」许可证
- 洞察：每个人都有一张「以后再XX」的清单，但「以后」永远不靠站
- 碰撞结果：「以后偷走了你好多个今天」

### 3.6 动词的用法

动词不一定要驱动行为，可以造画面/意境。
- 「领取春日礼」→ 不是去拿实物，是把阳光花香当作礼物接收
- 「捕到一张live图」→ 不是真的捕，是偶然发现的惊喜感

## 第四层：创作陷阱 — 避开什么？

### 4.1 Jarvis 陷阱五维（AI / 工程师的通病）

| 维度 | 陷阱（错的） | 范本（对的） |
|------|------------|------------|
| 开头 | 挖痛点 / 制造焦虑 | 重新定义 / 轻快切入 |
| 引导 | 建议动作（1）2）3） | 展示画面（有人在X/有人在Y） |
| 角色 | 我（作者）在说话 | 让真实用户的声音出现 |
| 语气 | 运营口吻（公告腔） | 朋友凑过来说（私聊感） |
| 结尾 | 中规中矩收口 | 让读者感觉被珍视 |

**最核心的一句话差距：**
> ❌ 我在教读者「怎么做」
> ✅ 我在展示「什么样的人已经在做」
> 前者是攻略，后者是范本。

### 4.2 AI 腔四类

| 类别 | 特征词 | 替换方向 |
|------|--------|---------|
| 汇报化 | 提升/优化/赋能/闭环/颗粒度 | 变得更好/改一改/帮你/从头到尾/细节 |
| 模板化 | 首先其次最后/综上所述/众所周知 | 想到几件事/大概就是这些/删掉 |
| 懂行感 | 据相关数据显示/研究表明/底层逻辑 | 我看到一个数字/有研究说/其实就是 |
| 教读者 | 你应该/你必须/划重点/答案是 | 你可以试试/我自己是/我后来发现 |

### 4.3 算法红线

- #AI生成内容 必标（2026/01 起强制）
- 标题前 13 字含核心关键词
- 话题 2泛+2垂+1长尾
- 单日 ≤ 2 篇，间隔 ≥ 2h

## 第五层：创作系统 — 怎么持续进步？

### 5.1 渐进式教练（一次只改一维）

看出问题 ≠ 写得出来。一次收到 6 维反馈 = 信息过载。
**一次只 focus 当前最差的那一维**，改完再看下一维。

### 5.2 多切面法则

> 「同一个 slogan，六个切面，六种受众。文案本身不变，不同人群看到的是自己想看的那一面。这不是缺陷，是力量。」— Allen

每篇笔记独立成文，不靠前后连载。同一个主题换不同角度再写一次——敢于自我推翻，才是创作。

### 5.3 功夫积累

> 「练拳不练功，到老一场空。」— 东东枪 097
> 「跟世界不熟的人写不出打动世界的好句子。」— 东东枪 098

- **术（技巧）**：句式、结构、标题公式 → 可以速成
- **道（理论）**：AB 点、核心体验、洞察 → 需要理解
- **功（内力）**：眼界、审美、分寸感 → 只能积累

积累方式：多看案例、多拆解爆款、多写多改、场景库持续扩容。

### 5.4 场景库维护

AI 写文案最大短板：场景库太窄，反复用同一批词。每次写作前：
1. 从场景库挑 3 个没用过的画面
2. 不要同一类目挑 3 个（避免单一感官堆砌）
3. 用过的画面在档案里标记，下次避开

## 决策速查：写一篇笔记的 8 个问题

动笔前：
1. **心象：** 我为谁写？（不是人口标签，是认知状态）
2. **AB 点：** 她现在怎么想？看完想让她怎么想？
3. **核心体验：** 我卖的不是内容，是哪种感觉？
4. **功课：** 如果涉及品牌/产品，我查了官方定位吗？

动笔中：
5. **留白：** 这段话是把答案给她了，还是把空间留给她了？
6. **五法：** 有没有出现配料/成分/参数/功能描述？有就改掉。

写完后：
7. **Jarvis 陷阱：** 我是在教「怎么做」还是在展示「什么样的人」？
8. **AI 腔：** 删掉所有「提升/优化/赋能/众所周知/首先其次最后」。

## 实战对照：从攻略型到范本型的完整转变

以下是一篇防晒霜笔记从「烂」到「好」的转变过程，每一轮对应一层哲学。

### 原始稿（攻略型，Jarvis 陷阱全中）

> 夏天到了，你是不是也为防晒头疼？今天分享三招：
> 1. 选 SPF50+ 的防晒霜
> 2. 每两小时补涂一次
> 3. 回家一定要卸妆
> 记住这三点，夏天白成一道光！

**诊断：** 开头挖痛点 ✓ 教读者怎么做 ✓ 作者在说话 ✓ 运营口吻 ✓ 中规中矩结尾 ✓

### 第一轮：用心象定义读者（第一层）

> 如果你也是「一晒就黑」的那种人，你不是一个人。
> 我去年整个夏天都在试，终于找到了一套不费力但管用的法子。

**改了什么：** 不挖痛点（「一晒就黑」是事实陈述，不是制造焦虑），「你不是一个人」= 归属感。

### 第二轮：设 AB 点（第二层）

> A（读者现在）：防晒 = 麻烦，要记时间、要补涂、要卸妆
> B（看完后）：防晒可以是一个很温柔的日常仪式

> 我后来发现，防晒这件事从「任务」变成「仪式」之后，就再也没漏过。

**改了什么：** 不是教怎么防晒，是重新定义防晒——从麻烦变成仪式。

### 第三轮：用五法改写（第三层）

> ❌ 选 SPF50+ 的防晒霜
> ✅ 那支防晒挤出来像融化的奶油，涂开有一种被薄纱轻轻罩住的感觉

> ❌ 每两小时补涂
> ✅ 下午三点，阳光斜进窗户，我从包里摸出那支小银管，像给皮肤按了一次暂停键

> ❌ 回家要卸妆
> ✅ 晚上洗脸的时候，水混着防晒流下来的那一瞬，觉得今天又被好好对待了一次

**改了什么：** 配料→感受，功能→场景，产品→情绪，成分→过程，描述→比喻。

### 第四轮：查 Jarvis 陷阱（第四层）

> 再检查：开头是重新定义（不是挖痛点）✓
> 正文是展示「有个人把防晒活成了一个温柔的日常仪式」（不是在教步骤）✓
> 语气像和朋友分享自己的发现（不是运营公告）✓
> 结尾让读者觉得「我也可以这样对自己好一点」（不是中规中矩收口）✓

### 最终稿

> 防晒这件事，我今年终于不再跟它较劲了。
>
> 以前总觉得防晒=麻烦——记时间、补涂、卸妆，三步少一步都不行。
> 后来换了一支防晒，挤出来像融化的奶油，涂开有一种被薄纱轻轻罩住的感觉。
>
> 现在出门前涂它，不是因为「应该」，是因为那个质感让我觉得我在对自己好。
> 下午三点阳光斜进来的时候，从包里摸出那支小银管，像给皮肤按了一次暂停键。
>
> 晚上洗脸，水混着防晒流下来——嗯，今天又被好好对待了一次。
>
> 你说防晒是护肤的最后一步，还是对自己好一点的第一步？

**转变总结：**

| 维度 | 原始稿 | 最终稿 |
|------|--------|--------|
| 开头 | 挖痛点「你是不是也…」 | 分享真实感受「不再跟它较劲」 |
| 内容 | 列步骤 1）2）3） | 展示三个场景，每个都有画面 |
| 角色 | 教别人的专家 | 分享发现的普通人 |
| 语气 | 公告+教程 | 朋友私聊 |
| 结尾 | 喊口号「白成一道光」 | 留一个问题让读者接 |
| 字数 | 80 字 | 180 字 |
| AI 腔 | 严重 | 无 |

## 常见借口（别信）

| 借口 | 现实 |
|------|------|
| 「道理我都知道」 | Jarvis 陷阱——Allen 说过「认知到位了，输出还没跟上」。知道 ≠ 写出来。 |
| 「这次简单，不用跑哲学检查」 | 最简单的文案最容易犯 AI 腔。30 秒检查永远值得。 |
| 「我先写出来再改」 | 攻略型思路写出来的东西，后面改的是文字不是思路。从头就用范本型。 |
| 「读者看得懂就行了」 | 读者看得懂 ≠ 读者有共鸣。共鸣驱动 CES（评论×4/转发×4/关注×8）。 |
| 「我写的不是商业文案，不用 AB 点」 | 个人笔记也有 A→B——你每篇都在改变读者对某件事的看法。 |
| 「时间紧就不查功课了」 | 不查品牌官方定位 = 从零发明产品 = 大概率 0 分。查功课只要 5 分钟。 |
| 「这个意象之前用过但这次合适」 | 场景库窄的症状。去 scene_library 换 3 个没用过的画面。 |

## 第六层：创作能量与持续力 — 怎么不被掏空？

> 前五层是「写得对」的方法论。第六层是「写得久」的方法论。
> 大多数人不是技法不够，是**持续 30 天后不知道怎么续命**。这一层是写给「写到第 60 天的你」。

### 6.1 写不出 ≠ 技法问题

```
写不出来的三种真实原因（按出现频率排序）：
  1. 能量低 ─ 没睡好 / 情绪堵塞 / 生活没有真实输入
  2. 状态僵 ─ 在分析模式里出不来（看了 20 篇爆款研究）
  3. 才是技法问题 ─ 但概率 < 20%
```

**别用技法解决能量问题：** 看更多教程 / 拆更多爆款 / 改更多框架 = 在能量已经为负的时候继续抽空。
**先把能量补回来**：散步 30 分钟、关掉手机睡一觉、做一顿饭、给自己写一段不发的文字。

### 6.2 没有灵感的日子怎么产出（节奏 > 灵感）

> 「专业作者不等灵感，灵感等专业作者。」— 创作老话

灵感型创作者写得高峰期很猛，低谷期断更几周。**节奏型创作者**有应对低谷的预案：

| 状态 | 不要做 | 要做 |
|---|---|---|
| 高峰期（灵感喷涌） | 写完就发 | 写 3 篇 → 留 2 篇做「储粮」 |
| 中等期（普通日子） | 强行追爆款 | 用上次储粮里的那篇，今天补一个 B 点更新 |
| 低谷期（写不出） | 硬憋 / 不发 / 自责 | 用「续写」「整理」「短点评」型笔记保住更新节奏 |

**储粮型笔记**（任何状态都能写）：
- 「今天看到的一个细节」（30 字以内 + 1 张图）
- 「这周我重新理解了一个词」（解释一个旧词的新含义）
- 「问你一个问题」（纯互动型，不输出观点，邀请评论）

### 6.3 发出去的最后 10%（克服删除冲动）

写完到发出去之间有一个「**审判区间**」——你会突然觉得这篇配不上你，想删掉重写。

> 这种感觉**不是质量信号，是暴露感**。
> 你不是觉得它写得差，你是觉得「让人看见这一面的我会不会很傻」。

**对治方法：**
1. **24 小时缓冲**：写完后**不要当场发也不要当场删**，存草稿，第二天上午再看。
2. **量级降级法**：如果还是想删，问自己「能不能换个标题、删 20% 后发？」——保住产出节奏比保住单篇完美重要。
3. **关掉自我编辑**：发布后**别再看自己的笔记**至少 6 小时。盯着数据看 = 把创作能量换成焦虑。
4. **删 vs 发的判断标准**：
   - ✅ 发：让你脸红但说的是真话 / 不完美但有一句真挚
   - ❌ 删：抄了别人的洞察 / 写完没共鸣自己都不信

### 6.4 低谷-高峰循环：把它当作正常生理

```
创作者的真实曲线（不是直线进步）:
高峰 ──╲    ╱──╲    ╱──╲      ← 高峰会回来
       ╲  ╱    ╲  ╱    ╲
低谷    ╲╱      ╲╱      ╲    ← 低谷不是终点
        ↑       ↑       ↑
       2 周    1 周    3 天
       (低谷期会越来越短)
```

**低谷期的 3 个允许：**
1. 允许产出降速（每周 5 篇 → 2 篇是健康调整）
2. 允许风格漂移（这周写得不像「我的风格」也没关系，可能在长出新的一面）
3. 允许重复主题（同一件事换 3 个角度写，不是水文，是把它咀嚼透）

**低谷期最有效的事：去看不写小红书的人在做什么**（去美术馆、菜市场、夜班公交、老同学聚会）。
**输入**才是低谷期的解药——但不是去看更多笔记，是去看真实生活。

### 6.5 持续力的反面：耗尽信号

提早识别耗尽，比熬到崩溃修复成本低 10 倍。

| 早期信号 | 中期信号 | 晚期信号（必须停） |
|---|---|---|
| 写完不想再读一遍 | 看到爆款只剩嫉妒 | 写一句话身体开始反胃 |
| 凑数发不挑剔了 | 数据焦虑超过创作快乐 | 看到自己的账号头像就累 |
| 选题前先看竞品 | 模仿成主导，原创成例外 | 怀疑自己为什么开始做这件事 |

**早期信号出现：** 调节作息 + 减少 30% 产出。
**中期信号出现：** 停 1 周不发，只读不写。
**晚期信号出现：** 停 1 个月，找一个完全无关的事情做（学游泳 / 下厨 / 见 5 个老朋友）。

### 6.6 续命的 3 个长期工具

1. **创作日记**（不是文案，是给自己看）：每天 100 字写「今天我感受到的一件事」。
   作用：保住「感受能力」不退化。**感受能力是文案的根**。
2. **同行者圈子（不是同行）**：找 1-3 个**也在长期写**的人定期聊（不一定要写小红书）。
   作用：缓解「是不是只有我这样」的孤立感。
3. **每季一次「不为发布的写作」**：写一篇你**确定不会发**的笔记，纯粹为了写。
   作用：把创作的快感从「数据反馈」剥离回「写本身」。
