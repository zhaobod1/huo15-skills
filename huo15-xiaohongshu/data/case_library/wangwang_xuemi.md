# 旺旺雪米饼：先做功课再动笔

## 品牌官方信息

- **官方广告语**：「好米匠心，旺旺真够味」
- **产品核心**：以「米」本身为主角
- **官方定位**：好米匠心 = 态度，不是成分
- **产品卖点**：「米本来的味道」而非「没有添加剂」

## 关键认知

- 「好米匠心」是一种做事的态度，不是对原料的描述
- 卖的是「米本来的味道」，不是「没有添加剂」
- 出发点错 → 全篇白费

## 失败版本分析（F~H，0分）

| 版本 | 问题 |
|------|------|
| F | 对白太直白，文字无美感，自嗨 |
| G | 普及知识/介绍产品，不是情绪 |
| H | 「雪米饼没有声音」违背事实，咔嚓脆才是卖点 |

## 60分文案五法

1. **用感受代配料** — 不说成分，说吃下去的感觉
2. **用场景代功能** — 不说能干嘛，说在什么场景下吃
3. **用情绪代产品** — 以读者情绪为主角，非产品
4. **用过程代成分** — 写从稻田到舌尖的旅程
5. **用比喻代描述** — 用自然意象包裹产品

## Allen 核心意象

纯粹米香、谷物原生、温柔治愈、松弛感

## 苏格拉底式问题

1. 写产品文案前，你做了哪些功课？
2. 品牌的官方定位是态度还是成分？你怎么判断？
3. 你上一稿里出现了哪些「配料/参数/功能」？换成感受会怎样？
4. 「咔嚓一声脆」和「雪米饼没有声音」，哪个是对的？为什么对错很重要？
5. 为什么「先查再写」比「想好了再写」更安全？

## 范本解读

Allen 的三篇 60 分文案不是「写得特别好」，而是「完全不在错误的频道上」。
他先研究了旺旺的官方定位，找到了「好米匠心」这个态度的锚点，然后所有意象（米香/原生/治愈/松弛）都从锚点出发。
贾维斯的版本 F~H 之所以 0 分，不是因为文字差，是因为出发点就错了——从零发明产品，跟品牌频道完全脱离。

## 你能学到的

- **行动顺序比创意能力更重要**：先查后写 > 先想后查
- **品牌自己在说什么 > 你觉得品牌应该说什么**
- **态度 vs 成分**：这是两个完全不同的赛道，跑错赛道全篇白费
- **事实核实**：写「没有声音」之前，先咬一口确认是不是咔嚓脆
