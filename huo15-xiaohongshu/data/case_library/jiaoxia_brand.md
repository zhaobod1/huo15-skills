## 原文 / 范本（蕉下经典文案）

> 不是害怕太阳，而是拥抱太阳。
> 人不是去了户外，而是回到了户外。
> 风是最好的造型师。
> 像仙人掌一样，给肌肤加一层防晒层。
> 每一口呼吸，都是松弛的味道。

## 关键技法

- **句式一：不是 X，而是 Y** — 颠覆旧认知，重新定义价值
- **句式二：修辞 / 比喻** — 把功能寄生在自然意象上（参数变画面）
- **句式三：每一 X 都是 Y** — 把日常颗粒拉高成仪式感
- **品牌核心体验：** 不卖防晒衣，卖"敢晒太阳的野心"

## 苏格拉底问题（先你自己回答）

1. **「不是害怕太阳，而是拥抱太阳」中，"害怕太阳"是真的事实吗？为什么蕉下要这样写？**

2. **「风是最好的造型师」这句话隐藏了什么产品功能？为什么不直接说"防风性能优秀"？**

3. **「每一口呼吸都是松弛的味道」对应蕉下的什么产品特性？为什么这样翻译？**

4. **如果你做一个"早C晚A"护肤品牌，蕉下三式怎么用？**

5. **蕉下的"核心体验"是"敢晒太阳的野心"。读者真的买走"野心"吗？还是买走"能放心出门的安心"？**

---

## 范本解读

### 1. "害怕太阳"是真的吗

不一定是事实，但**是消费者隐含的心理**。
夏天出门防晒、戴帽、撑伞 — 这些动作潜台词都是"对太阳的防御态度"。
蕉下没有发明新真相，**他们发现了一个被遗忘的真相**：
我们其实希望出门是开心的事，不是设防的事。

这就是东东枪说的"**洞察 = 未被发现或已被遗忘的真相**"。

### 2. "风是最好的造型师" 的产品翻译

明面上：把"抗风"参数变成自然意象。
深层：把消费者的"功能需求"（不被风吹乱）换成"美学体验"（被风做造型）。

**为什么不直接说"防风性能"：**
- 参数 / 数字 / 技术词 — 读了就忘
- 意象 / 比喻 — 直接进入读者脑中的画面，留存远超 5 倍
- "防风"是产品角度；"造型师"是用户角度 — 用户中心。

### 3. "每一口呼吸" 翻译的是什么

对应蕉下面料的**轻量化 / 透气性**。
直接翻译会是："面料透气性极好，不闷"。
蕉下式翻译是："呼吸都不被打扰 → 呼吸是松弛的 → 你穿着是松弛的"。

**升维路径：**
功能（透气）→ 体感（不闷）→ 状态（松弛）→ 价值观（松弛是种生活态度）

每一层都比上一层更让消费者认同，但又都是从同一个产品功能延伸出来的。

### 4. 早 C 晚 A 用蕉下三式

```
不是 X 而是 Y：
"不是补救熬夜的脸，而是给皮肤一个早餐和晚安"

修辞 / 比喻：
"维 C 是清晨第一杯水"
"A 醇是给皮肤的睡眠"

每一 X 都是 Y：
"每一次涂抹，都是和明天的脸打个招呼"
"让每一次照镜子都变得期待"
```

注意：好的蕉下式不是套模板，是**先想清楚核心体验**，再选最贴的句式表达。

### 5. 卖"野心"还是卖"安心"

**两者都是**，但层次不同：
- "安心"是核心**功能体验**（防晒做到位 → 不晒黑 → 安心）
- "野心"是品牌**精神体验**（敢出门 → 重新定义"出门" → 价值观认同）

普通品牌卖功能体验就够了。
**品牌型品牌**卖精神体验 — 因为只有精神体验能产生"溢价"和"忠诚"。

---

## 你能学到的

1. **写品牌文案前先找洞察**：不是发明，是发现"被遗忘的真相"
2. **三式不是套模板**：先想清楚"核心体验"，再选最贴的句式
3. **功能 → 体感 → 状态 → 价值观** 升维路径，每段文案都该走完一段
4. **比喻要"参数寄生"**：把抽象功能寄生到具象自然意象上
5. **品牌卖的是"野心"层级**：功能层级是入门，精神层级才能溢价
