# 上班就是不合理的超自然现象：金句共鸣法

## 原文要点

核心只有一句话：「上班就是不合理的超自然现象。」

没有解释为什么，没有论证，没有「三大原因」——就这一句，独立成段。

## 关键技法：金句法

> 「好文案不是写出来的，是留出来的。」— Allen

**金句法的三个条件：**
1. **反常识** — 正常人都觉得「上班是正常的」，你说「不合理的超自然现象」
2. **不解释** — 金句独立成段，后面不跟「因为」「这是因为」「原因有三」
3. **情绪共识** — 读者看到后会心一笑，自己往里填情绪

**为什么不能解释：** 一解释，就从「共鸣」变成了「说教」。读者本来在点头，你一解释她就开始分析你对不对。

## 结构拆解

```
金句（1 句，独立成段）
  ↓
留白（不解释，让读者自己的情绪填进去）
  ↓
共鸣产生（「哈哈哈真相了」= 评论×4）
```

全文可能只有 50 字，但 CES 高于 500 字的攻略文。

## 苏格拉底式问题

1. 你最近有什么想说的，能用一句反常识的话表达吗？
2. 为什么「不解释」比「解释」更有力？
3. 这种金句法适合什么赛道？不适合什么赛道？
4. 如果你的金句不够反常识，怎么改？

## 范本解读

这句话的力量来自三重反差：
- **上班（日常） vs 超自然现象（异常）** — 概念碰撞
- **不合理的（价值判断） vs 所有人都在做的事（社会规范）** — 戳破共识
- **一句话完事（形式） vs 大家都在论证上班的意义（语境）** — 留白

## 你能学到的

- **金句不要铺垫**：直接扔出来，让它在读者脑子里炸
- **反常识是钩子**：你不需要制造焦虑，只需要换个角度说人人都在做的事
- **短就是力量**：50 字的金句 > 500 字的论证
- **停止解释**：你觉得需要解释的地方，通常是你还不够自信的地方
