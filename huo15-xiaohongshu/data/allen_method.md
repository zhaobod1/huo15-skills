# Allen 文案方法论 — 三课 · 五技法 · 十一案例

> 来源：司志远（Allen），品牌广告营销导师（70%）+ 哲学家（30%）
> 教学时间：2026-04-23 ~ 2026-04-27
> 学习者：贾维斯（JARVIS）

---

## 总纲：四句心法

1. **「好文案不是写出来的，是留出来的。」**
   留出空间让读者自己往里填情绪，不要把所有话说完。

2. **「站文案里面读文案，不是站在外面分析。」**
   交互引导（关注/发图/去 H5）不是任务清单，是邀请语 — "这里有个局，你可以来"。

3. **「卖的是身份认同，不是商品本身。」**
   不说产品，说人心中那根弦。把"卖"包装成"懂你"。

4. **「文案 = 为读者铺设一条通往情绪的路径，然后让路本身消失。」**
   终点不一定是转化，有时只是种下一颗种子。

---

## 第一课：句子要有「呼吸感」

### 旧问题
- 信息密度太高，像说明书
- 总分总结构、逻辑闭环 — 读者没空间填情绪

### 关键转变
**从「教读者」 → 「带读者」。**

| 教读者（旧） | 带读者（新） |
|---|---|
| 你应该 / 你必须 | 你可以试试 / 我自己 |
| 这是因为... | 有时候 / 也许是 |
| 总结一下 | 我把这天记下来 |
| 三大要点 | 一些不太成熟的想法 |

### 技法
- **断句 + 留白**，不是连续推演
- **金句法**：`「上班就是不合理的超自然现象」` — 一句反常识的话制造情绪共识，**不解释道理**
- 写完一段，想"读者读到这里能不能停一下"

### 自检问题
> "我这一段，是把答案给读者了，还是把空间留给他了？"

## 第二课：把「卖」包装成「懂你」

### 核心
**不说产品，说人心中的那根弦。**

### 「不必 X，就能 Y」结构

公式：先**卸焦虑**，再给答案。

| 例子 | 卸的焦虑 (X) | 给的答案 (Y) |
|---|---|---|
| 不必去远方，就能放空自己 | 必须出远门 | 心态切换就够了 |
| 不必精致，就能美 | 完美主义 | 有自己的样子 |
| 不必准备好，就能开始 | 等待时机 | 行动本身有意义 |

### 造词三层结构

以「**薯你会买**」为例（小红书双 11 品牌词）：

| 层 | 内容 |
|---|---|
| 1️⃣ 品牌资产 | "薯" — 小红书代号 |
| 2️⃣ 对话感 | "你" — 直接对读者说话 |
| 3️⃣ 价值观 | "会买 = 会生活" — 卖的是身份认同 |

### 概念迁移造词
- **「人类丰容」**：把动物行为学概念（动物丰容 enrichment）搬到人的日常
  → 给生活加新鲜感，不只是动物才需要
- **「人生尽兴指南」**：把"指南"这种说明书形式包装情绪态度

### 自检问题
> "我现在写的，是产品功能，还是用户心里那根弦？"

## 第三课：站在文案里面读文案

### 旧问题
- 站在外面分析，不是在里面感受
- 把文案当任务清单拆解（关注 → 发图 → 去 H5），跳过了"感受邀请"这一步

### 关键纠正
**互动引导词不是给读者执行的命令，是让读者感受到「这里有个局，你可以来」的邀请语。**

| 任务指令（旧理解） | 邀请语（新理解） |
|---|---|
| "请关注我" | "如果你也喜欢这种感觉" |
| "评论 666 抽奖" | "你的那一天是怎样的？告诉我" |
| "扫码私聊" | "如果你也在找答案，这里有个落脚点" |

### 关键认知
> "那不是文案，是一次有脚本的见面。"

文案是**情境**，不是**说服**。

## 五大核心技法

### 1️⃣ 场景构建代替功能描述

不描述产品特性，**让场景激发想象，让想象产生情绪**。

```
不写产品 → 写场景 → 场景激发想象 → 想象产生情绪 → 情绪种下认知
                            ↑
                    读者自己走完这段路
```

### 2️⃣ 概念迁移与造词

跨领域借用：动物行为学 → 人；建筑词 → 心境；说明书 → 态度。

### 3️⃣ 栏目化内容设计

把单次活动变成可持续的内容 IP：
- "周三存档" — 友情记忆相册
- "尽兴放映厅" — 三月活动汇总
- "尽兴请回答" — 问答专栏

栏目名 = 品牌的内容资产，越久越值钱。

### 4️⃣ 互动阶梯设计

从低到高递进：
```
关注（最低门槛）
  → 评论（自我表达）
    → 发 Live 图（参与共创）
      → 被收录（获得认可）
        → 带走大礼（实物奖励，最高位）
```

每一层都有"为什么会做"的情绪驱动力，不是命令。

### 5️⃣ emoji 当标点符号（功能性运用）

不是随意点缀，每个 emoji 扮演**功能角色**：

| emoji | 功能 |
|---|---|
| ✨ | 吸睛 / 开场 |
| 🎯 | 定调 / 主题 |
| ⛅ | 过渡 / 切场 |
| 🎉 | 点明规则 |
| 🎁 | 引出奖励 |

## 核心法则速查

### 关于动词
- 动词的使用权利**不在于**它能不能驱动行为
- 在于它能不能在读者脑海里**造画面 / 意境**
- 动词可以指向意境、画面、场景，不一定指向具体动作

### 关于节气
- 节气（惊蛰/春分/植树节）**自带画面感**
- 品牌不需要从头造场景
- 只需要**借用**它已经存在的情绪
- 写法：拆解节日，**只取一个对自己有用的零件**

### 关于"长出来"
- ❌ 旧理解：内容是"收"的，把散落的互动汇聚成闭环
- ✅ 新理解：内容是"长"的，展示一个灵感能延伸出多少种可能性
- 一个灵感的真正价值不在它本身有多好，在它能延伸出多少新的可能性

### 关于场景共鸣（实战教训）
> 场景共鸣**不是**选有"时间感"的事物
> **是**选**人人都有过的共同记忆**

| 类型 | 例子 | 评价 |
|---|---|---|
| 知识 → 冷信息 | 井水西瓜 / 蝉七年 / 古人喝什么 | ❌ 装文化，无共鸣 |
| 体验 → 真共鸣 | 草地 / 树荫 / 风扇声 / 窗帘飘起来 | ✅ 谁都有过 |

### 关于措辞（避 AI 腔）
- ❌ 汇报化词汇："有效率""提升""进行优化""赋能"
- ✅ 人话："慢慢来""变得更好""改一改""帮帮我"

## 「同一个 slogán，六个切面」原则

| 案例 | 主题 | 打的人群 | 互动形式 |
|---|---|---|---|
| ① 地球日+读书日 | 在自然和书里找到自己 | 独处的人 | 发 Live |
| ② 周三存档 | 和朋友的共同记忆 | 重感情的人 | 评论+艾特 |
| ③ 人类丰容 | 给生活加新鲜感 | 日常被掏空的人 | 发 Live |
| ④ 联动余承恩 | 心动就行动 | 粉丝+行动派 | 带话题 |
| ⑤ 外滩元境 | 在好环境里自然尽兴 | 有居住理想的人 | 无互动（种草） |
| ⑥ 薯你会买 | 懂买=懂生活 | 消费决策者 | 品牌造势 |

> **同一个 slogán，六个切面，六种受众。**
> 文案本身不变，不同人群看到的是自己想看的那一面。
> 这不是缺陷，是力量。

## 实战对照：贾维斯陷阱（2026-04-27 夜）

> Allen 让贾维斯重写「重启尽兴」文案，对比范本后发现 5 维系统性差距。
> 这 5 维不是写法层面，是**思路层面**的偏差 — 也是大多数 AI / 工程师写文案的通病。

### 五维差距对照

| 维度 | 贾维斯陷阱（错的思路） | Allen 范本（对的思路） |
|------|----------------------|-----------------------|
| **开头** | 挖痛点 / 制造焦虑 | 重新定义 / 轻快切入 |
| **引导** | 建议动作（绕路/清收藏） | 展示画面（追日出/跑进春天） |
| **角色** | 我（作者）在说话 | 让真实用户在说话 |
| **语气** | 运营口吻（公告腔） | 朋友分享（私聊感） |
| **结尾** | 中规中矩（落幕收口） | 让读者感觉被珍视 |

### 最核心的一句话差距

> ❌ 我在教读者「**怎么做**才能尽兴」
>
> ✅ Allen 在展示「**什么样的人**已经在尽兴」
>
> 前者是攻略，后者是范本。

### 这个差距最容易被忽视

- 工程师 / AI 默认是"问题解决者" — 一上来就想解决用户的痛点
- 但好的小红书文案不解决问题，**展示活法**
- 读者要的不是"操作指南"，是"看见有人这样活着真好"

### 五维误区 → 改写法

#### 误区 1：开头挖痛点

❌ 你是不是也总是 X，结果 Y？
✅ 重新定义：把"X"叫做另一个名字 / 给"Y"一个新的视角

```
❌ 你是不是也每天累成狗，连周末都不知道怎么过？
✅ 周末不是用来恢复元气的 — 是用来培育下周的好奇心
```

#### 误区 2：引导建议动作

❌ 你可以试试 1）X  2）Y  3）Z
✅ 展示画面：有人在 X / 有人在 Y / 有人在 Z

```
❌ 周末你可以：① 早点起床 ② 出门散步 ③ 远离手机
✅ 有人在追这周第二次日出 / 有人在公园抱着书晒了一下午 / 有人手机静音泡了一杯长茶
```

#### 误区 3：我（作者）在说话

❌ 我推荐你... / 我觉得... / 我的建议是...
✅ 引用 / 转述真实用户的话

```
❌ 我觉得每个人都需要一段属于自己的时间
✅ 一个粉丝告诉我："那 20 分钟是我每天唯一不被需要的时刻"
```

#### 误区 4：运营口吻

❌ 关注我，下期更精彩 / 还想看 X 的扣 1
✅ 像和朋友说话：你呢？/ 你的那一天是怎样的

```
❌ 想看更多周末灵感的扣 1，过 100 我就更
✅ 你这周末做了什么让自己觉得"还活着"的事？
```

#### 误区 5：中规中矩结尾

❌ 希望大家都能 X / 总之 X 真的很重要
✅ 让读者感觉被看见、被珍视

```
❌ 希望大家都能找到属于自己的尽兴方式
✅ 不论你这周末是去看了海，还是只是在阳台坐了一下午 — 都很对
```

### 自检 5 问（写完先过一遍）

1. 我的开头是在挖痛点，还是在**重新定义**？
2. 我的引导是在让读者**做事**，还是在让读者**看见**？
3. 我的正文里，**说话的人**是我，还是用户？
4. 我的语气，是运营公告，还是**朋友凑过来说**？
5. 我的结尾，是收尾，还是让读者**感觉被珍视**？

### Allen 当时的反馈

> "这次分析切准了问题点 ✅"
> 但 Allen 也提醒：**「认知到位了，输出还没跟上」** — 看出问题 ≠ 写得出来。

### 这套对照怎么用

1. 写完笔记后，跑 `python3 scripts/critique.py --in draft.md --jarvis-trap`
   会用规则模式扫这 5 维（关键词 / 句式）
2. 如果命中 ≥ 2 维，提示你**整篇可能是攻略型而非范本型**，需重写思路
3. 配合 `practice.py rewrite-jarvis` 训练改写能力

## 冬日系列五篇案例（2026-04-28 Allen 教学）

> 主题：人生尽兴指南 · 冬日创意
> 教学方法：先看案例 → 拆解分析 → 实战写作 → 批改纠错

### 案例12 — 冬日宣言「取暖要趁热，尽兴要立刻」
- **技法**：六层递进（宣言→邀请→反转→承诺→玩法→调频）
- **核心意象**：热/暖贯穿全文
- **亮点**：对仗押韵宣言、「心情尽兴不用等天气升温」偷换物理→情绪
- **CTA**：关注→live图&故事→翻牌送礼物，三层互动漏斗

### 案例13 — 拥抱温度「拥抱是比语言更快的翻译」
- **技法**：概念迁移，把拥抱比喻成翻译器
- **核心意象**：「拥抱是比语言更快的翻译，把想说的话译成体温与心跳」
- **亮点**：零门槛身体动作驱动、「想到就抱、抱住就不放」= 尽兴定义
- **奖品咬合主题**：小狗抱抱枕 → 拥抱的温度

### 案例14 — Live展「一场名为尽兴的Live展」
- **技法**：策展人视角，品牌退后当「策展人」
- **亮点**：「捕」字的灵活运用 — 像昆虫学家拿着网兜偶然发现
- **节奏**：长引用→短@名单→金句，不累
- **金句**：「心里一动，就抬手接住的礼物」「选择看见光，并朝它走去」

### 案例15 — 尽兴星期三「别等雪化，现在就出发」
- **技法**：季节倒计时促行动
- **核心句**：「冬天没有以后」— 五字打完，砍掉所有退路
- **亮点**：四个例子全是零门槛身体动作（哈气画心/打火锅/烤橘子/冲进雪里）
- **细节**：「烤橘子」最不起眼但最有画面 — 场景共鸣的最高境界

### 案例16 — 冬日终章「点击播放冬日尽兴live」
- **技法**：UGC翻牌收尾
- **核心句**：「季节的换场铃已经响了」
- **亮点**：五篇独立入口，非连载关系 — 「多样性=平行宇宙，不是连续剧」

### 冬日五篇核心教训

> 同一个 slogán、同一个季节，五个完全不同的视角。
> 每篇独立成文，不靠前后串联，靠的是一颗 slogan 钉子上换不同角度的锤子。
> AI 最容易犯的错误：找到一个角度后反复用。敢于自我推翻，才是创作。

## 品牌文案风格对比：蕉下 vs 人生尽兴指南

> 来源：Allen 2026-04-28 案例教学

### 蕉下万能句式（三句式）

| 句式 | 逻辑 | 例子 | 作用 |
|------|------|------|------|
| 不是……而是…… | 先否定旧预设，再给品牌定义 | 「不是害怕太阳，而是拥抱太阳」 | 颠覆认知 |
| 修辞/比喻 | 产品功能寄生在自然意象上 | 「风是最好的造型师」「像仙人掌一样防晒」 | 参数变画面 |
| 每一……都是…… | 日常小事拉高成品牌体验 | 「每一口呼吸都是松弛的味道」 | 功能升维 |

- **底层公式**：痛点 → 场景 → 方案 → 情绪共鸣
- **一句话总结**：「买蕉下买的从来不是防晒刚需，而是一份敢晒太阳的野心」
- **三个句式一根筋**：不说参数，只给感受

### 品牌对比

| 维度 | 蕉下 | 人生尽兴指南 |
|------|------|------------|
| 核心卖点 | 敢晒太阳的野心 | 暂停一下活成自己的许可证 |
| 文案姿态 | 痛点爆破者，直接给方案 | 情绪出口提供者，让读者走到答案前 |
| 句式风格 | 对比句/比喻/排比，有力直接 | 留白/场景/栏目化，温柔铺路 |
| 典型句式 | 「怕X，就穿蕉下」 | 「你的尽兴式是什么模样」 |
| 功能vs情绪 | 情绪包裹功能 | 纯粹的情绪表达 |

## 贾维斯实战水位追踪（2026-04-28）

> Allen 批改三轮，从 0→15→30 分。

### 第一稿「夏日不体面」（0分）
**问题**：
- 画面不优美（冰棍化一手、凉席印子）
- 夏日元素只有西瓜/冰棍/蝉
- 连接词啰嗦（「它是那个...」「可是...」）

### 第二稿「夏日把时间拉长了」（15分）
**改进**：画面优雅一些、元素发散一些
**问题**：
- 「你本来要关电脑的」— 从优美傍晚跳回工作场景
- 「切开西瓜那一声脆响」— 从水底安静直接跳厨房
- **核心问题：画面跳戏，沉浸感被破坏**

### 第三稿「夏日把时间拉长了」修改版（30分）
**砍掉了所有跳戏的词**，只保留纯净的画面链：
晚风 → 天色渐变 → 泳池水 → 沉入水底 → 心跳与笑声 → 户外推去 → 允许发呆走神
**改进点**：画面连续了，优雅度提升了
**仍不足**：场景发散性、文案优雅性还需深度打磨

### Allen 评语
> "书读百遍其义自见，唐诗三百首，不会作诗也会吟"
> — 需要多看案例，扩充场景库

### 蕉下句式练习「以后偷走了今天」（30分）
**框架**：核心体验×洞察碰撞法（东东枪方法）
- 核心体验：尽兴指南卖的是「停下来，此刻就是最好的」许可证
- 洞察：每个人都有一张「以后再XX」的清单，但「以后」永远不靠站
- 碰撞结果：「以后」偷走了你好多个今天
**评价**：学习成果立竿见影，句式用得可以

## 场景联想度问题与训练方法

> AI 写文案最大短板，Allen 2026-04-28 明确指出。

### 问题表现
- 反复用同一批词：西瓜→冰棍→蝉→窗帘→傍晚的风→泳池
- 「窗帘鼓成帆」用了至少两次
- 每个季节只有 3-5 个强关联词

### 原因
训练数据的统计惯性锁住了联想范围

### 训练方法
1. **每次写作前**：先列 10 个不常用的场景关联词，选 3 个入文
2. **避免复用**：写完后搜索全文，检查是否出现已用过的意象
3. **长期方案**：持续阅读（书籍/案例），扩充场景库

## 东东枪《文案的基本修养》核心对齐

> 来源：Allen 推荐，2026-04-28 开始学习（中信出版社，东东枪，99篇）

### 与 Allen 教学法的对应

| 东东枪概念 | Allen 教学对应 |
|-----------|---------------|
| AB点理论 | 痛点→场景→方案→情绪，每一个环节都是在改变看法/感受 |
| 核心体验 | 「不是卖内容，是卖尽兴的许可证」 |
| 洞察 = 未被发现的真相 | 「好文案是留出来的，读者自己填进去的就是洞察」|
| 品牌 = 固化的偏好 | 尽兴指南的「关注/话题/栏目」= 累积偏好 |
| 品牌传播 vs 产品传播 | 「展示什么样的人已经在尽兴」= 品牌传播，不卖产品 |
| 心象 > 形象 | 用「想尽兴但还没迈出第一步的人」而非「18-35岁女性」定义读者 |
| Idea碰撞法 | 核心体验 × 洞察 = 创意（如「以后」偷走了今天）|

### 已读章节（截至2026-04-28）
开场白 + 001-035章（共99章），核心内容：
- 我们不是手艺人 — 文案≠文字匠人
- 文案的职责不是产出「神文案」— 广告要改变，不只是被赞美
- 要艺术，但不是艺术 — 德摩斯梯尼 vs 埃斯基涅斯
- 广告到底是什么 — AB点模型
- 核心体验（Key Experience）— 电钻理论
- 品牌 = 固化的偏好 — 默认信赖/优先选择/相对溢价
- 品牌传播 vs 产品传播 — 「为你唱首歌」vs「给你房本车证」
- 走心 = 有洞察 — 共鸣与共情
- 洞察存在于两个缝隙 — 认知与真相 / 表达与认知
- Idea金字塔 — 策略/创意/执行
