# SKILL.md - huo15-memory-evolution

> 火一五记忆进化技能 — 长期记忆自我进化系统

---

## 核心概念

记忆不是静态存储，而是动态演化的生命体。本技能参考 Claude 的 Memory Evolution 方案，设计三层记忆架构：

```
┌─────────────────────────────────────────┐
│           记忆进化系统架构                │
├─────────────────────────────────────────┤
│                                         │
│  ┌─────────────┐  ┌─────────────────┐  │
│  │ 工作记忆     │  │ 上下文记忆       │  │
│  │ (Session)   │  │ (Context)       │  │
│  │ 当前对话    │  │ 近期交互摘要     │  │
│  └─────────────┘  └─────────────────┘  │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ 长期记忆 (MEMORY.md)             │   │
│  │ - 人物关系                       │   │
│  │ - 项目状态                       │   │
│  │ - 重要决策记录                   │   │
│  │ - 偏好与习惯                     │   │
│  └─────────────────────────────────┘   │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ Reference 记忆                   │   │
│  │ - 系统指针                       │   │
│  │ - 工具使用规范                   │   │
│  │ - 项目参考                       │   │
│  │ - 知识库同步                     │   │
│  └─────────────────────────────────┘   │
│                                         │
└─────────────────────────────────────────┘
```

---

## 三层记忆

### 1. 工作记忆（Session Memory）
- 当前对话的临时上下文
- 对话结束自动归档到 daily notes

### 2. 长期记忆（MEMORY.md）
- 人物关系图谱
- 项目状态追踪
- 重要决策记录
- 偏好与习惯
- 每小时自动整理一次

### 3. Reference 记忆
- 系统级参考文档
- 可被知识库技能同步

## 核心脚本

| 脚本 | 功能 |
|------|------|
| `memory-evolve.sh` | 主进化脚本，每天定时运行 |
| `memory-compact.sh` | 压缩归档旧记忆，提取精华 |
| `memory-sync.sh` | 与知识库双向同步 |
| `memory-audit.sh` | 体检记忆文件，修复损坏 |

---

## 使用方法

```bash
# 每日记忆进化（建议 cron 每天执行）
memory-evolve

# 手动整理
memory-evolve --compact

# 体检
memory-evolve --audit

# 与知识库同步
memory-evolve --sync
```

## 进化规则

### 遗忘规则
- 30 天未访问的记忆条目 → 移入归档
- 重复矛盾的记忆 → 保留最新，标记冲突
- 情绪记忆（负面）→ 降低权重但保留

### 强化规则
- 被反复引用的记忆 → 提升权重
- 重要决策关联的记忆 → 长期保留
- 用户明确指定"重要" → 永久保留

### 整合规则
- 同主题多条记忆 → 合并摘要
- 项目结束 → 归档并更新项目状态
- 人物新信息 → 更新人物卡片
