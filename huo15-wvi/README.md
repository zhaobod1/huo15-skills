## 简介

**火一五工作价值观量表**（`huo15-wvi`）是基于 Donald E. Super《Work Values Inventory》开发的 HR 一对一调研技能。

通过 **一问一答** 的对话形式，五步走采集员工的真实价值偏好：

```
①17项快速评分 → ②追求的点 → ③不愿放弃的点 → ④保留三项核心 → ⑤三项满足度评分
```

最终自动生成：

- 📊 **价值观全景**（17 项排名 + 内在/外在/报酬结构偏向）
- 🎯 **追求的点**（员工最想从工作中得到什么）
- 🚩 **不愿放弃的底线**（触碰即流失的红线）
- ⭐ **核心三项 + 满足度**（重要 × 满足落差的 Gap 分析）
- 💡 **HR 洞察简报**（人物画像 + 流失风险预警 + 三条建议）

> 理论支撑见 [`docs/WVI理论报告.md`](docs/WVI理论报告.md)。

## 17 个价值维度

| # | 维度 | 核心含义 |
|:---:|---|---|
| 1 | 经济报酬 | 薪酬满意度 |
| 2 | 成就感 | 目标达成与正向反馈 |
| 3 | 智慧 | 持续学习与认知提升 |
| 4 | 能力运用 | 技能发挥空间 |
| 5 | 生活方式 | 工作与生活平衡 |
| 6 | 社交关系 | 人际网络广度 |
| 7 | 上司关系 | 与领导相处融洽度 |
| 8 | 同事与团队 | 团队协作感 |
| 9 | 名声地位 | 社会认可与地位 |
| 10 | 工作环境 | 物理环境舒适度 |
| 11 | 安全感 | 职业稳定性 |
| 12 | 自主性 | 工作自控度 |
| 13 | 利他助人 | 社会价值感 |
| 14 | 审美 | 美学追求 |
| 15 | 多样性 | 新鲜感与多元接触 |
| 16 | 冒险与挑战 | 不确定性接受度 |
| 17 | 创造力 | 创新与突破空间 |

## 使用方法

```
# 直接说：
工作价值观调研
# 或
WVI量表
# 或
开始价值观测评
```

技能触发后，AI 调研助手会自动开场并按五步引导。  
全程约 **5 分钟**，完成后自动输出含流失风险预警的 HR 报告。

## 评分规则

| 分值 | 含义 |
|:---:|---|
| 1 | 对我完全不重要 |
| 2 | 对我不太重要 |
| 3 | 一般重要 |
| 4 | 比较重要 |
| 5 | 对我非常重要 |

- **跳过** / **不确定** → 自动记为 3 分（中性）
- **我想改第 X 题** → 支持返回修改，修改后从当前位置继续
