## 一、量表起源

| 项目 | 内容 |
|---|---|
| 提出者 | 唐纳德·舒伯（Donald E. Super，1910–1994），美国生涯发展理论奠基人 |
| 量表名 | 工作价值观量表（Work Values Inventory, WVI） |
| 首版年份 | 1970 年，由 Houghton-Mifflin 出版发行手册 |
| 原始结构 | **15 个价值维度**，每维度 3 道题，共 **45 题**，**5 点李克特量表** |
| 修订版 | 1989 年 Super & Nevill 在「工作重要性研究」（Work Importance Study, WIS）框架下推出价值观量表（Values Scale），扩展了维度；后续多国本土化版本维度数有差异（13–21 项不等） |

WVI 是生涯辅导与职业咨询领域的奠基性工具，核心用途是**帮助个体把"个人价值观"与"职业选择"对齐**，从而提升职业的持久度与满意度。

---

## 二、理论内核：价值观为何决定职业满意度

舒伯认为，职业选择是**自我概念（self-concept）的实现过程**。一个人在工作中追求的，本质是他认为"重要、值得、能定义自己"的东西。

- **价值观 ≠ 兴趣**：兴趣是"喜欢做什么"，价值观是"为什么而做、要从中得到什么"。
- **匹配 → 满意 → 留任**：当工作能满足一个人的核心价值观时，满意度与忠诚度显著上升；反之即便薪酬高，核心价值长期得不到满足也会离职。
- **价值观有优先级**：人不会"什么都要"，真正起作用的是**排在前面的少数几项**——这正是本技能"保留三项"设计的理论依据。

## 三、三大价值结构（WVI 核心分类）

舒伯把工作价值归为三大类，这是解读结果最重要的框架：

| 维度 | 含义 | 典型动机 |
|---|---|---|
| **内在价值**（Intrinsic） | 与工作**本身性质**有关，做事过程本身带来的满足 | 成长、创造、贡献、自我实现 |
| **外在价值**（Extrinsic） | 与工作的**环境与人际**有关，工作"周边"的条件 | 稳定、舒适、关系融洽 |
| **外在报酬**（Concomitant / 附带报酬） | 工作**带来的回报与身份**，工作的"结果" | 金钱、地位、生活品质 |

> **HR 解读要点**：
> - 偏**内在价值** → 用成长、挑战、意义感激励，给自主权与作品署名。
> - 偏**外在价值** → 重视稳定与团队氛围，慎用高压变动，保障环境与关系。
> - 偏**外在报酬** → 薪酬、晋升、头衔是核心杠杆，绩效—回报链路要清晰可见。

## 四、17 项价值维度详解（本技能采用版）

本技能采用国内生涯教育常见的 **17 项本土化扩展版**，与舒伯原始维度对照，并归入三大结构：

### 内在价值（9 项）

| 维度 | 解释 | 对应原始维度 |
|---|---|---|
| 成就感 | 不断取得成就、获得积极反馈、达成目标 | Achievement |
| 智慧 | 工作中可不懈追求真理、提升智慧 | Intellectual Stimulation 智力激发 |
| 能力运用 | 能发挥自己希望使用的能力 | Ability Utilization（WIS 扩展） |
| 自主性 | 能自主控制工作进度、内容、时间 | Independence 独立性 |
| 利他助人 | 对他人和社会有价值、有益 | Altruism 利他主义 |
| 审美 | 不断追求美的东西、获得美的享受 | Aesthetics |
| 多样性 | 能接触新鲜的、不同的人与事物 | Variety |
| 冒险与挑战 | 工作充满冒险和挑战、有不确定性 | Risk / Challenge（WIS 扩展） |
| 创造力 | 能独创性地提供新观点、项目、构架，不循规蹈矩 | Creativity |

### 外在价值（5 项）

| 维度 | 解释 | 对应原始维度 |
|---|---|---|
| 社交关系 | 能和各种人交往，建立广泛的社会联系 | Social Interaction |
| 上司关系 | 同领导相处愉快、自然 | Supervisory Relations |
| 同事与团队 | 以团队形式开展工作并实现团队目标 | Associates / Co-workers |
| 工作环境 | 追求舒适、轻松、优越的工作条件 | Surroundings |
| 安全感 | 工作中有一个安稳的局面 | Security |

### 外在报酬（3 项）

| 维度 | 解释 | 对应原始维度 |
|---|---|---|
| 经济报酬 | 工作能提供满意的经济报酬 | Economic Return |
| 名声地位 | 工作能带来想要的名声和社会地位 | Prestige 声望 |
| 生活方式 | 职业有喜欢的生活方式，如工作与生活平衡 | Way of Life |

> 合计：**内在 9 + 外在 5 + 报酬 3 = 17 项**。

## 五、评分与解读方法

### 1. 评分规则（5 点量表）

| 分值 | 含义 |
|:---:|---|
| 1 | 对我完全不重要 |
| 2 | 对我不太重要 |
| 3 | 一般重要 |
| 4 | 比较重要 |
| 5 | 对我非常重要 |

### 2. 解读三步

1. **看排名**：分数从高到低，前几项 = 核心驱动力，决定择业/留任。
2. **看结构**：把高分项归入内在/外在/报酬三大类，看偏向哪一类。
3. **看落差（Gap）**：核心价值"重要度高 + 当前工作满足度低" = **流失风险点**，这是 HR 最该关注的信号。

### 3. 解读注意事项

- **无对错**：价值观没有高低优劣，只有匹配与否。
- **看相对不看绝对**：很多人会把多数项打高分，**真正有意义的是相对排序**，所以本技能强制"保留三项"做取舍。
- **情境敏感**：价值观会随人生阶段（如成家、升职）变化，建议**定期复测**（如年度）。
- **不可单独定生死**：测评是参考，需结合绩效、访谈、实际行为综合判断。

## 六、HR 应用场景

| 场景 | 用法 |
|---|---|
| **招聘选才** | 判断候选人核心价值与岗位/团队文化是否匹配，降低错配离职 |
| **入职引导** | 新人入职 30 天内测一次，主管据此定制带教与激励方式 |
| **留任与激励** | 识别核心价值"想要但没满足"的落差，针对性补救，预防流失 |
| **团队配置** | 同岗不同价值取向的人搭配，平衡稳定型与挑战型 |
| **晋升与转岗** | 评估调整后的岗位能否继续满足其核心价值 |
| **离职面谈** | 反推流失的核心价值缺口，沉淀为组织改进点 |

## 七、本技能（huo15-wvi）的实现映射

本技能在标准 WVI 评分之上，增加了三个对 HR 极有价值的环节：

| 环节 | 理论依据 | HR 产出 |
|---|---|---|
| ① 17 项快速评分 | WVI 标准 5 点量表 | 价值观全景与三大类偏向 |
| ② **追求的点** | 价值观优先级（高分项即驱动力） | 员工最想从工作中得到什么 |
| ③ **不愿放弃的点** | 价值观取舍测试（底线识别） | 触碰即可能离职的红线 |
| ④ **保留三项** | "真正起作用的是少数核心项" | 择业/留任的三大核心标准 |
| ⑤ **三项满足度评分** | 重要度 × 满足度落差（Gap 分析） | 流失风险预警 + 激励抓手 |
