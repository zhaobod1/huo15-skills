# 火一五文档技能

> 国家公文标准 Word 文档生成 — 青岛火一五信息科技有限公司

## 核心能力

输入纯文本内容，自动生成符合 **GB/T 9704-2012 国家公文格式标准** 的 Word 文档：

| 要素 | 标准 |
|------|------|
| 页面边距 | 上 3.7cm，下 3.5cm，左 2.8cm，右 2.6cm |
| 正文字体 | 仿宋，小四（12pt），1.5 倍行距，首行缩进 2 字符 |
| 一级标题 | 黑体，三号（15pt），加粗，居中 |
| 二级标题 | 楷体，四号（12pt），加粗 |
| 页眉 | 左对齐，LOGO + 公司名称，中间空一格，底边细线 |
| 页脚 | 居中，格式：**第 X 页 / 共 Y 页** |
| 内容格式 | 纯文本，自动识别标题段落，不保留任何 markdown 符号 |

## 快速使用

### Python 代码调用

```python
from create_word_doc import create_word_doc

# 最简用法：标题 + 正文
create_word_doc(
    output_path="合同_客户名_20260408.docx",
    title="销售合同",
    content="""一、甲方信息
甲方的名称、地址、联系人。

二、乙方信息
乙方的名称、地址、联系人。

三、合同金额
本合同总价为人民币壹佰万元整。
"""
)
```

### 命令行调用

```bash
python create-word-doc.py <输出文件> [标题] [正文]
python create-word-doc.py 报告.docx "项目报告" "一、项目背景..."
```

## 页眉页脚说明

### 页眉
- **来源**：自动从公司系统（Odoo）获取 LOGO 和公司名称
- **布局**：左对齐，LOGO 在左，公司名称紧跟其后，中间一个空格
- **底线**：页眉下方有一条黑色细线分隔

### 页脚
- **格式**：`第 X 页 / 共 Y 页`
- **居中**显示，仿宋字体

### 无公司系统时的降级
- LOGO：从 `https://tools.huo15.com/uploads/images/system/logo-colours.png` 下载
- 公司名：默认使用「青岛火一五信息科技有限公司」

## 内容格式规则

### 段落类型自动识别

| 开头 | 识别为 | 字体 |
|------|--------|------|
| 一、 二、 三、... | 二级标题 | 楷体，加粗 |
| 第X章 / 第X节 | 一级标题 | 楷体，加粗 |
| （一） （二）... | 三级标题 | 仿宋，加粗 |
| 其他文字 | 正文段落 | 仿宋，首行缩进 |

### 表格支持
- 使用 `|` 分隔，例：`| 列1 | 列2 |`
- 第一行自动识别为表头，黑体居中

### 禁止出现以下内容
- `**`、`#`、`-`、`` ` `` 等 markdown 符号
- `[文字](链接)` 格式
- 管道符 `|`（表格外）

> ⚠️ 内容输入前，请去除 markdown 语法符号，只保留纯文本。
